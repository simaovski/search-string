  • How to use?

To use the Re-searcher you should the follow steps:
  • First, select the library;
  • Second, write the first term that you desire research;
  • Third, click in button to add a new term;
  • Fourth, write the new term;
  • Fifth, repeat the steps (add and write term) up until complete the terms of your search string

You should copy the result of search string and realize three differents searches in selected library. For each search, the Re-searcher build a specific search string. Thus, you should copy the content in TITLE SEARCH card and paste in selected library. After this, you should perform again the search with ABSTRACT and KEYWORD cards, respectively.

This varies according the library selected. But, in this version, the Re-searcher the not have a limite in number of terms for add in your search string. In future, the Re-seacher will be adapted follow the rules in each library supported.

The Re-searcher are in your first version. Because of this there a limited number of libraries supported. However, the libraries supported are the very important in many fields of knowledge and are vast areas of research for any researcher.
