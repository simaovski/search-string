About Re-searcher

Learn more about the Re-seacher project and who is involved.

About the project: The focus of this project is generate one application that help researchers build the search string in your works of systematic review or mapping.

  • Context: Systematic mapping and review studies are forms of review the scientific literature about a determined theme. In many cases, this is a first step for write a master dissertation or doctoral thesis, locating the researcher about the current state of art.
  • Motivation: But, many researchers have doubts about how to build a search string to conduct the revision process. Furthermore, there no many mechanisms to auxilie the student in this task.
  • Objective: Because of that, this project was created. The main aim of Re-searcher is facilitate researches in systematic studies, automatizing the construction of search string.
